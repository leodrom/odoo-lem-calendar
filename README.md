# Календар Подій — LEM Calendar Module

Кастомний модуль календаря подій для Odoo 19, розроблений для [LEM Station](https://lemstation.com).

## Можливості

- **Єдиний календар** — відображає події, CRM-угоди та ручні записи в одному місці
- **Локації** — прив'язка подій до локацій (Площа, Фудмаркет, Дах, Двір, Клуб тощо) з кольоровим маркуванням
- **Статуси** — кожен запис має статус із візуальним відображенням (емодзі на картці календаря, кольоровий badge)
- **Пов'язані об'єкти** — запис може бути прив'язаний до події (`event.event`) або CRM-угоди (`crm.lead`)
- **Фільтри** — фільтрація по локації, типу запису та статусу в sidebar календаря
- **Chatter** — повний лог змін через `mail.thread` + `mail.activity.mixin`

## Структура модуля

```
lem_calendar/
├── models/
│   ├── lem_calendar_entry.py   # Основна модель запису календаря
│   ├── lem_location.py         # Модель локацій
│   └── event_event.py          # Розширення event.event
├── views/
│   ├── lem_calendar_entry_views.xml  # Calendar / List / Form views
│   ├── lem_location_views.xml        # Views для локацій
│   └── menus.xml                     # Меню
├── static/
│   ├── description/icon.png    # Іконка модуля
│   └── src/css/calendar.css    # Стилі (radio-кнопки статусів, popover)
├── security/ir.model.access.csv
└── i18n/uk_UA.po
```

## Встановлення

1. Скопіюйте модуль у папку `custom_addons`
2. Оновіть список модулів в Odoo
3. Встановіть "Календар Подій"

**Залежності:** `mail`, `event`, `crm`

## Технічні деталі

- **Odoo версія:** 19.0
- **Ліцензія:** LGPL-3
- **Кольори локацій:** стандартний `widget="color_picker"` (12 кольорів, Integer 0–11)
- **Іконка меню:** оновлюється тільки через ORM (`ir.ui.menu.write({'web_icon': ...})`), не через прямий SQL
